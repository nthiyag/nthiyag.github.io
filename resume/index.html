Holonomic Pure Pursuit Simulator

Jan 2021

  • Motivation
    • It had been a longtime personal goal of mine to implement Pure Pursuit type path following algorithms for autonomous control of VRC robots (see VRC)
    • During Covid lockdowns, I was unable to test these algorithms on a physical VEX robot
    • Instead, I decided to simulate them virtually so that I could implement them on a physical robot later (unfortunately I graduated before lockdowns were lifted)
  • Developed a Python based simulator using a Tkinter frontend to simulate the first order dynamics of a holonomic drivetrain (using high school physics concepts)
  • Implemented a path planner and Pure Pursuit controller on this virtual drivetrain, assuming perfect odometry input
  • Given an initial and target robot pose, the algorithm first interpolates between them using a Hermite spline (a cubic), then samples a number of discrete waypoints along the spline
  • At each step, the drivetrain follows a simple proportional control loop targeting one of these waypoints - once it gets close enough, it switches the target to the next waypoint along the spline
  • This process repeats, while angular error is handled by a separate proportional control loop (can be decoupled since the system is holonomic)
VEX COTS Spacing Calculator

May 2020

  • Motivation
    • When assembling mechanisms for VRC (see VRC), improper spacing in the assembly of VEX COTS parts can lead to flimsiness or a drop in performance (usually due to friction)
    • This is normally aleviated by using CAD software (ie. Solidworks) to plan out the relative spacing of each part, but that is unwieldy when rapidly prototyping to test design ideas
    • This leads to prototypes being flimsy/inefficient, which provides a skewed understanding of design ideas
  • Developed a Tkinter based application in Python to aid in rapidly calculating the optimal spacing between VEX COTS parts
  • This part spacing tool allows users to virtually test out combinations of COTS spacing elements between standard VEX parts - this is much faster than full blown CAD, and requires no physical guesswork
